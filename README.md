# Daily Log — Claude Code Skill

5-minute daily energy-mood-focus journal for Claude Code. Creates structured daily logs in your `notes/` folder.

## Installation

```bash
git clone https://github.com/bayramannakov/daily-log-skill.git ~/.claude/skills/daily-log
```

## Usage

Type `/daily-log` in Claude Code. The skill will walk you through 6 quick questions (energy, focus, mood, highlight, blocker, tomorrow) and save a structured markdown log.
